# SQL 

SQL (ang. Structured Query Language) to język zapytań, za pomocą którego możemy komunikować się z silnikiem bazodanowym. Za pomocą zapytań SQL możemy m. in.:
- tworzyć nowe tabele,
- modyfikować tabele,
- usuwać tabele,
- pobierać dane,
- filtrować dane,
- grupować dane,
- sortować dane.
